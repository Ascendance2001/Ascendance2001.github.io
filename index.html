The Clean Divorce

in 2023-2024 I woked on a project for the company "The Clean Divorce"

The Clean Divorce is a company based in Ontario who specializes in helping people go through Divorce. Such as giving them Medation sessions and similair. I, with 7 other people, remade their website from scratch using Wix. We chose Wix because of it's fast prototyping capabilities and addon library allowing us to build common features such as the shop, or profile pages instantly so we can prioritize on the hard much more custom solutions.

One of these custom solutions is where I spent the most of my time. It's called Journey to Freedom. The idea of Journey To Freedom is to split up traditional medation sessions into small easily digestable chunks called stages which a customer can go through at their own pace. This also saves them money as Journey to Freedom is much cheaper then these medation sessions.

Each stage is a single site which draws from a database to dynamically populate new content that includes videos and lots of very useful text based on a topic of a divorce. Each stage also has a "gratitude questionare" at the bottom which is tracked then graphed when a customer makes it through all 8 stages of the Journey to Freedom.

Youtube Jams Discord Bot

in 2023 I woked on a personal project creating a Discord bot

Tired of Google going on a rampage shutting down any music bots ability to play music from Youtube, I learned how to create my own. Meet Youtube Jams. Youtube Jams is a private Discord bot made in Java using JDA for the main Discord api, Lavaplayer for playing audio, yt-dlp for downloading Youtube videos (Don't tell Google), and Spotify Web API Java for Spotify support.

The bot works by taking in a command with JDA. in the case of a Spotify link uses the Spotify api to figure out the name, artist, and album of the song, Searches if nessacary and downloads the video off of Youtube using yt-dlp. Then plays the resulting webm file with Lavaplayer after a song finishes playing the webm file is deleted so the bot never takes up more space then it needs.

Puzzlemaker Plus

this is a project I started in 2023

Puzzle Maker Plus is an ambitious project being worked on by me, vrad.exe (Kelsey), SP2G5000, and TeamSpen210. We are going to make, from the ground up, an external level editor for the game Portal 2 by Valve.

The vanilia, ingame, Puzzlemaker is good, however it's very clunky the smallest voxels that can be edited are huge. twice the height of the player, at least. It also is not mod-able at all: Beemod tries to mod Puzzlemaker but there are so many limitation to work past that are hard coded into the game. Such as no cusomizablity at all to context menus. An external editor can get around all of this because we can make it modular ready for modding for the start. We can also fix all the annoyance that "vanilia" Puzzlemaker has. And then add our own features like multiple people collabarating on a map at once. If they are ok with it we can also embed it into the upcoming mod Portal 2: Community Edition which does not have any puzzle maker in it at all right now.

Puzzle Maker Plus is going to be made in C++ with Qt for user interface and hud (heads up display) elements. The 3D map editing view will be rendered using OpenGL
