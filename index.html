The Clean Divorce

In 2023-2024, I worked on a project for the company "The Clean Divorce"

The Clean Divorce is a company based in Ontario who specializes in helping people go through divorce. They give them mediation sessions and similar. I, with 7 other people, remade their website from scratch using Wix. We chose Wix because of it's fast prototyping capabilities and addon library allowing us to build common features such as the shop, or profile pages instantly so we can prioritize on the hard and much more custom solutions.

One of these custom solutions is where I spent the most of my time. It's called Journey to Freedom. Journey To Freedom is to split up traditional mediation sessions into small easily digestible chunks called stages which a customer can go through at their own pace. This also saves them money as Journey to Freedom is much cheaper than these mediation sessions.

Each stage is a single web page which draws from a database to dynamically populate new content that includes videos and lots of very useful text based on a topic of a divorce. Each stage also has a "gratitude questionare" at the bottom which is tracked then graphed when a customer makes it through all 8 stages of the Journey to Freedom.

YouTube Jams Discord Bot

In 2023, I worked on a personal project creating my first Discord bot

Tired of Google going on a rampage shutting down any music bots ability to play music from YouTube, I learned how to create my own. Meet YouTube Jams. YouTube Jams is a private Discord bot made in Java using JDA for the main Discord api, Lavaplayer for playing audio, yt-dlp for downloading YouTube videos and for Soundcloud support, and Spotify Web API Java for Spotify support.

The bot works by first taking in a command with JDA. In the case of a Spotify link, it uses the Spotify api to figure out the name, artist, and album of the song. Then it searches if necessary and downloads the video off of YouTube using yt-dlp. It then plays the resulting webm file with Lavaplayer after a song finishes playing the webm file is deleted so the bot never takes up more space than it needs.

Puzzlemaker Plus

In 2023, I started work on an external level editor for Portal 2

Puzzle Maker Plus is an ambitious project being worked on by me, vrad.exe (Kelsey), SP2G5000, and TeamSpen210. We are going to make, from the ground up, an external level editor for the game Portal 2 by Valve.

The vanilla, in game, Puzzlemaker is good, however it's very clunky the smallest voxels that can be edited are huge. They are almost twice the height of the player. It also is not mod-able at all: Beemod tries to mod Puzzlemaker but there are so many limitation to work past that are hard coded into the game. Such as no customizable at all to context menus or no way to expand the amount of items that can be added a time. An external editor can get around all of this because we can make it modular and ready for modding from the start. We can also fix all the annoyance that vanilla Puzzlemaker has. And then add our own features like multiple people collaborating on a map at once. We are also in talks with the Portal 2: common Edition team to embed into their upcoming mod which does not have any form of puzzle maker in it right now.

Puzzle Maker Plus is going to be made in C++ with Qt for user interface and hud (heads up display) elements. The 3D map editing view will be rendered using OpenGL
